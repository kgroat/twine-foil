Foil passages have access to a helper function called 'show'.

Using this function, you can navigate to another passage within the same story. For instance, let's say you have a passage called 'dayTwo'. You can go from the current passage to the new passage in the following manner:

The sun goes down, and you find yourself growing tired.
<% $('#sleep').click(function(){ show('dayTwo') }) %>
