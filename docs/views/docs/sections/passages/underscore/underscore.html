Foil also comes with Underscore available for use.

In your passages, you'll have access to a variable called '_' which will be an instance of Underscore.

Inventory:
<% var items = ['1x T-Shirt', '1x Pair of Slacks', '2x Sock']; %> <% _.each(items, function(item){ %> You have: <%= item %>
<% }) %>

Underscore is a very powerful tool that allows you to manipulate lists in many different ways.

Inventory:
<% var items = [{ name: 'T-Shirt' },{ name: 'Pair of Slacks' },{ name: 'Sock' },{ name: 'Sock' }]; %> <% _.chain(items) .groupBy(function(item){ return item.name; }) .map(function(list, name){ return list.length + 'x ' + name; }) .each(function(item){ %> You have: <%= item %>
<% }) %>

For more information on Underscore, you should look at its documentation.
