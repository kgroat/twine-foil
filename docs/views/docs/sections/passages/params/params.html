  Foil passages can allow parameters to be passed between them.

This means that if you have a variable in one passage, when you call the show() method, you can pass in an object as the second parameter. The object passed in will be available in the shown passage using the variable 'params'.

To make a parameter required, you can add a tag to the passage: 'required-X' where you replace the X with the name of the required variable. For instance, if you want to make a parameter called 'character' to be required, you would add the 'required-character' tag.

If you try to show a passage with a required parameter without supplying the parameter, you will be shown an error.

Here, we have two passages called 'left' and 'right' respectively, each of which requires a parameter called 'character':

<% var character = { name: 'Christina' } %>
Which way do you go?
<% $('#go-left').click(function(){ show('left', { character: character }) }) $('#go-right').click(function(){ show('right', { character: character }) }) %>
