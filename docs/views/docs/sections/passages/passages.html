Passages in Foil use a templating pattern similar to Snowman that allows you to write JavaScript embedded directly in your text. Unlike Snowman, however, Foil does not use Markdown syntax. All html must be written by hand, and is interpreted as-is.

You can use the syntax &lt;% ... %&gt; to execute any JavaScript statement while the passage is being rendered. Simply replace the contained ... with the statement you would like executed, and it will be run when the passage is rendered. If you would like the results of the JavaScript statement to be printed into the passage, use the &lt;%= ... %&gt; syntax instead.

<% var sayHello = function(name) { return 'Hello, ' + name; } %> This is my greeting: <%= sayHello('world') %>
