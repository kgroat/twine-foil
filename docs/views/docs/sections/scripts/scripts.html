Scripts in Foil allow you to define reusable JavaScript helpers that can be used in your passages or within other scripts.

Scripts use CommonJS style modules, which allow you to define an object and have it be exposed by either setting it as module.exports or by setting properties on the exports variable.

You can create scripts by creating a passage and giving it the 'script' tag.

Here is an example of setting properties on the exports variable:

function Character(options){ this.name = options.name; } exports.asCharacter = function(character){ return new Character(character); };

You could also just export the Character constructor function directly:

function Character(options){ this.name = options.name; } module.exports = Character;
